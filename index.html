Fitness tracker has a login and register option on main page which takes you to the login and register page. The Register page is for the first time user to register his/her data in gym and start working out post it. Once the user register he/she needs to sign in as a user to continue to the routine/activity page.
After entering the routine page the user makes an entry in the routine page and feeds the information for the day which will mark as a record of what activity the user has done at which date and has left the gym at which time. There is a check progress button which will show the graphical growth of the user according to the wait. Routine page and info page contain the logout which logout the session and logo which will take you to the home page.
